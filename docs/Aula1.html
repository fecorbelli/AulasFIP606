Introdução ao R

O R é uma linguagem de programação estatistica gratuita e de livre acesso.

Pacotes no R

Um pacote é um grupamento de funções, dados e arquivos que podem ser utilizados no R. Há milhares de pacotes que são desenvolvidos e disponibilizados em repositórios para download e que podem nos auxiliar em nosso trabalho.

Instalação de pacotes

A instalação de pacotes pode ser feita pelo menu ou então com um comando no console. A instalação é feita através do menu em tools > install packages e digitando o nome do pacote para baixar ou usando a função instal.packages(nome do pacote) no console.

Após realizar a instalação do pacote, ele não estará disponível para uso imediatamente. Para habilitar o pacote para uso, se deve utilizar a função library(nome do pacote).

library(epifitter)

Objetos

Um objeto é um nome que armazena um valor. Para criá-lo utilizamos <- .

x <- 10
y <- x * 10

Conjunto de dados

Para criar um objeto que seja um conjunto de dados numéricos compreendidos em um intervalo, basta utilizar a função c(valor : valor).

a <- c(1:10)
b <- c(11:20)

Data-frame
